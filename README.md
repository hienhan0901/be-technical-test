# be-technical-test
## Clone project:
  ```sh
  git clone https://github.com/hienhan0901/be-technical-test.git
  cd be-technical-test
  ```
## Task 1:
  ```sh
  cd task1
  npm i
  npm start
  ```
## Task 2:
  Run back-end server (http://localhost:8000/).
  ```sh
  cd task2/task2-be
  npm i
  npm run dev
  ```
  Open a new terminal.
  Run front-end server (http://localhost:3000/).
  
  ```sh
  cd task2/task2-fe
  npm i
  npm start
  ```
